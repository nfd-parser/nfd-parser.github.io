# 什么是 NFD Parser

NFD Parser (netdisk-fast-download 网盘直链云解析) 是一个聚合多种主流网盘的直链解析下载服务，能够将网盘分享下载链接转化为直链，一键解析下载。

## 核心功能

- **多网盘支持**: 已支持蓝奏云、夸克网盘、UC网盘、小飞机盘、123云盘、移动云、天翼云等多款主流网盘
- **加密分享**: 支持加密分享链接以及部分网盘文件夹分享
- **认证解析**: 支持 Cookie/Token/用户名密码等多种认证方式解析大文件
- **高性能**: 基于 Vert.x 4 框架构建，支持高并发请求处理
- **缓存机制**: 智能缓存策略，提高解析速度
- **简单易用**: 提供简洁的 RESTful API 接口和短链格式

## 工作原理

1. **接收分享链接**: 用户提供网盘分享链接和可选的提取密码
2. **解析处理**: 服务器解析分享链接，获取文件信息
3. **生成直链**: 返回可直接下载的链接地址
4. **缓存优化**: 缓存解析结果，提高后续访问速度

## 技术特点

- **Java 17**: 基于现代 Java 开发（提供 JDK 11 分支）
- **Vert.x 4 框架**: 高性能异步处理
- **类 Spring 架构**: 注解式路由 API，开发友好
- **灵活配置**: 支持多种数据库、缓存方案和代理配置
- **容器化部署**: 支持 Docker 一键部署
- **JavaScript 解析器**: 支持 JS 编写自定义解析器扩展

## 使用场景

- **个人用户**: 简化网盘文件下载流程，命令行一键下载
- **开发者**: 集成到应用中提供网盘解析服务
- **企业**: 内部文件分享和管理
- **自动化工具**: 批量处理网盘链接
